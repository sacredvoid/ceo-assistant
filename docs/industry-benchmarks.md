# Industry Benchmarks Reference

Validation data for 13 industries. All figures are approximate and based on 2024-2025 market data.

**Data vintage:** Primarily sourced from IdeaProof.io (Jan 2026), supplemented by CB Insights, Carta, SaaS Capital, and industry reports. CAC channel benchmarks and valuation multiples shift annually - use `/ceo-validate` or `/ceo-compete` for live web research to supplement these baselines. Last updated: March 2026.

## 12. AI/ML

| Metric | Value |
|--------|-------|
| Market Size | $142B (projected $738B by 2030) |
| Growth Rate (CAGR) | 27% |
| Est. Success Rate | ~45% (higher for vertical AI, lower for infrastructure plays) |
| Avg Launch Cost | $50K-500K (fine-tuning/vertical), $1M+ (foundation model) |
| Time to PMF | 8-18 months (application layer), 18-36 months (infrastructure) |
| Typical CAC | $50-500 (B2B vertical AI), $5-50 (B2C consumer AI) |
| Typical LTV | $500-5K (SMB), $10K-100K (enterprise) |
| Failure Rate | ~55% within 3 years. Causes: no differentiation from big tech (30%), compute costs exceed margins (25%), no proprietary data moat (20%) |
| Fastest-Growing Segments | AI Agents (50%+ CAGR), Generative AI (45%), AI Security (40%), Vertical AI (35%), MLOps (30%) |
| Top Challenges | Compute costs, talent acquisition ($150-500K+/engineer), data quality, regulation (EU AI Act), differentiation from foundation model providers |

### Business Model Patterns

| Model | Description | Margin Profile |
|-------|-------------|---------------|
| Vertical AI App | Domain-specific solution (legal, healthcare, finance) | 60-80% gross margin |
| AI API/Platform | Inference or training API for developers | 40-60% gross margin |
| AI-Enhanced SaaS | Existing SaaS category with AI features | 70-85% gross margin |
| AI Agent/Workflow | Autonomous task completion (coding, sales, support) | 50-70% gross margin |
| AI Infrastructure | MLOps, data labeling, model serving | 40-60% gross margin |

### Compute Cost Benchmarks

| Workload | Cost |
|----------|------|
| Large Model Training (GPT-4 scale) | > $100M |
| Fine-Tuning (7-70B params) | $10K - $100K |
| Inference (per transaction) | $0.001 - $0.10 |
| Cloud GPU (A100/H100) | $1 - $4/hour |

### AI Startup Defensibility

| Moat Type | Strength | Example |
|-----------|----------|---------|
| Proprietary Data | Strong - compounds over time | Bloomberg Terminal, Scale AI |
| User-Generated Data Flywheel | Strong - more users = better model | GitHub Copilot, Grammarly |
| Vertical Domain Expertise | Moderate - hard to replicate quickly | Harvey (legal), Abridge (healthcare) |
| Distribution/Embedding | Moderate - switching costs | Salesforce Einstein, HubSpot AI |
| Model Architecture | Weak - replicable within months | Most open-source fine-tunes |
| Prompt Engineering/UX | Very Weak - zero defensibility | ChatGPT wrappers |

### Talent Costs

| Role | Annual Compensation |
|------|-------------------|
| ML Engineer | $150K - $400K |
| Research Scientist | $500K+ |
| AI Product Manager | $150K - $300K |
| Data Engineer | $130K - $250K |

### Funding Benchmarks

| Stage | Typical Round | AI Premium |
|-------|-------------|-----------|
| Pre-seed | $500K - $2M | 2-3x vs non-AI |
| Seed | $2 - $5M | 2x vs non-AI |
| Series A | $10 - $20M | 1.5-2x vs non-AI |
| Valuation Multiples | 10-50x ARR | Highest across all verticals |

### AI-Specific Metrics to Track
- Inference cost per query/transaction
- Model accuracy/quality score
- Latency (p50, p95, p99)
- Data pipeline freshness
- User retention by AI feature usage
- Cost per useful output (not just per API call)

## Cross-Industry Patterns

| Pattern | Details |
|---------|---------|
| LTV:CAC Ratio | 3:1 is the universal minimum target across all industries |
| Validation Impact | Proper pre-launch validation leads to 3x better odds of success |
| Sales Cycle Correlation | Consumer: 0-1 month, SMB: 1-3 months, Mid-Market: 3-6 months, Enterprise: 6-24 months |
| Bootstrap vs. Funded | Lower launch costs correlate with higher success rates in consumer categories |
| Regulatory Burden | Industries with heavy regulation (FinTech, HealthTech) have longer PMF timelines but higher LTV |
