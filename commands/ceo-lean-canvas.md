# Generate Lean Canvas

Create a 1-page Lean Canvas business plan.

## Input
Ask: "Describe your business idea. I'll fill in the 9 blocks and ask follow-ups for anything unclear."

## Process
1. Load template from @ceo-assistant/docs/business-templates.md (Section 1)
2. Fill each block in recommended order:

### Block 1: Problem (top 3 problems your customers face)
### Block 2: Customer Segments (target audience + early adopters)
### Block 3: Unique Value Proposition (single clear sentence of differentiation)
### Block 4: Solution (top 3 features addressing the top 3 problems)
### Block 5: Channels (how you reach customers)
### Block 6: Revenue Streams (pricing model, revenue sources)
### Block 7: Cost Structure (fixed + variable costs)
### Block 8: Key Metrics (3-5 KPIs that define success)
### Block 9: Unfair Advantage (what can't be easily copied)

3. For any block where info is insufficient, ask ONE focused question
4. Use WebSearch to validate market assumptions if needed

## Output
Present as a formatted grid, then offer:
- (a) Validate this with live market research (/ceo-validate)
- (b) Build financial projections from these assumptions (/ceo-calc)
- (c) Turn this into a pitch deck (/ceo-pitch-deck)
- (d) Generate a full business plan
